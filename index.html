Key Features

Everything you need

🧬

AI Synthesis

Claude API generates a unified personality narrative — finding convergences across all four frameworks and surfacing deep insights about identity, strengths, and relationships.

💾

Local-First Storage

All data lives in IndexedDB on your device. Works completely offline. Optional Supabase cloud sync for cross-device access — no data leaves without your consent.

🔀

Test or Skip

Already know your type? Every framework lets you skip the test and enter your known results directly. MBTI type selector, OCEAN sliders, Enneagram type picker.

🎨

Sleek Dark UI

Dark-mode-first design inspired by Co-Star and The Pattern. Radar charts for Big Five, SVG birth chart wheel, smooth card transitions and micro-animations.

⚙️

Robust State Machines

XState powers the multi-step test flows for reliability. Svelte 5 Runes handle reactive state elegantly. Compiled bundles stay tiny.

🌍

Multi-language Ready

Big Five IPIP-NEO items via @bigfive-org support 26 languages out of the box. Full internationalization architecture planned for all frameworks.

Quick Start

Up in 60 seconds

# Clone the repository
git clone https://github.com/yourusername/persona-synthesis.git
cd persona-synthesis

# Install dependencies
pnpm install

# Copy environment variables
cp .env.example .env.local

# Start development server
pnpm dev

# → Open http://localhost:5173

VITE_ASTRO_USER_ID / API_KEY optional

AstrologyAPI credentials for automatic natal chart calculation.

VITE_OPENCAGE_API_KEY optional

OpenCage geocoding for birth location lookup. 2,500 free requests/day.

VITE_SUPABASE_URL / ANON_KEY optional

Enable cross-device cloud sync and user authentication.

ANTHROPIC_API_KEY server-side

Powers the AI personality synthesis feature via Claude API. Never exposed client-side.
